# 🚢 Titanic Exploratory Data Analysis (EDA)

## 📌 Project Overview

This project performs Exploratory Data Analysis (EDA) on the famous Titanic dataset to uncover meaningful patterns, trends, and relationships between different passenger features.

The analysis includes statistical summaries, data cleaning, visualizations, and correlation analysis to better understand the factors affecting passenger survival.

---

## 🎯 Objectives

- Explore the dataset
- Handle missing values
- Analyze passenger demographics
- Understand survival patterns
- Visualize important relationships
- Perform correlation analysis

---

---

## 📈 Key Insights

- Most passengers were male.
- Female passengers had a significantly higher survival rate.
- Third-class passengers were the largest group.
- First-class passengers had the highest survival rate.
- Most passengers were between 20–40 years old.
- Higher ticket fares showed a positive relationship with survival.
